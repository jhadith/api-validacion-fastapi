# API Inventario y Ventas – Validación de Software (FastAPI)

Este repositorio contiene una API REST mínima para gestión de inventario y registro de ventas, junto con evidencias de validación mediante pruebas manuales y automatizadas (unitarias e integración), y métricas de cobertura.

## Requisitos
Python 3.11+

## Instalación
En la carpeta del proyecto, cree y active un entorno virtual e instale dependencias desde requirements.txt.

## Ejecución de la API
La API se ejecuta con Uvicorn y expone documentación Swagger en /docs.

## Ejecución de pruebas automatizadas
Las pruebas se ejecutan con PyTest, e incluyen pruebas unitarias y de integración.

## Cobertura
La cobertura se obtiene con Coverage y se reporta por consola.

## Evidencias
Las capturas, logs y checklist de aceptación se encuentran en la ruta docs/evidencias/.
